Giriş

WEB sitesi geliştirmek için bilmeniz gereken 3 tane dil vardır. Bunlar: HTML, CSS ve JS'dir

HTML

HTML sayfanın yapısını belirleyen bir işaretleme dilidir

Programlama dili değildir, sadece sitenin yapısı yazmak için kullanılır

zorunludur. İstersen CSS ve JS'i geçebilirsin ama HTML'i geçme şansın yoktur

HTML’de içerikleri etiketler ile oluşturursun. Her etiketin bir açılış (<) bir de kapanış (>) kısmı vardır

Temel HTML etiketleri

  • <h1> ... <h6> : Başlıklar
  • <p> : Paragraflar
  • <a> : Linkler
  • <img> : Resimler
  • <ul>, <ol<, <li> : Listeler

Örnek HTML kodu

                        
<h1>Merhaba Dünya!</h1>
<p>Bu bir paragraf.</p><a href="https://www.example.com">Örnek Link</a>

CSS

CSS, sayfanın görünümünü ve stilini belirleyen bir dildir. Renk, yazı tipi, boyut, boşluk ve diğer görsel özellikleri ayarlamak için kullanılır.

Örnek olarak bir başlığı kırmızı yapmak için CSS kullanabilirsin:

                        
h1 {
    color: red;
}

CSS’in sayfaya uygulanması 3 şekilde olabilir: inline, internal veya external. Bunlar HTML ile birlikte çalışır.

Temel CSS Özellikleri

  • color : Metin rengi
  • background-color : Arka plan rengi
  • font-size : Yazı boyutu
  • margin : Dış boşluk
  • padding : İç boşluk

JavaScript

JavaScript, sayfaya davranış ve etkileşim kazandırmak için kullanılan programlama dilidir. HTML ve CSS ile birlikte çalışır.

Örnek: Bir butona tıklandığında bir uyarı mesajı göstermek:

                        
function showAlert() {
    alert("Merhaba Dünya!");
}

HTML ile birlikte kullanımı:

                        
    <button onclick="showAlert()">Tıkla</button>

JavaScript sayesinde sayfada dinamik içerik oluşturabilir, kullanıcı etkileşimlerini yönetebilirsin.
